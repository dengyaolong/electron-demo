# A simple Markdown Editor By electron

## How to run

```
npm i electron -g 
git clone https://github.com/dengyaolong/electron-simple-markdown-editor.git
cd electron-simple-markdown-editor
npm install
npm start
```

## Package for Mac
```
npm i electron-packager -g
npm run build 
```

## Asar
```
npm i asar -g
npm run build 
```
